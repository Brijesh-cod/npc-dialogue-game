# NPC Dialogue Game (Python CLI)

A minimal command-line version of the NPC Dialogue Game converted from the web prototype.

## Features
- NPCs with mood, trust, memory
- Branching dialogue trees
- Inventory and item trading
- Quest offering based on trust

## How to run

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

The game runs in the terminal and stores state in memory while running.
